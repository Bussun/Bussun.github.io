Luma editor

A new BMG editor for Super Mario Galaxy. Now includes a graphical user interface! (More coming in future versions.)

Discover more! Downloads

Features (C# version)

  • Reading and editing message files from Super Mario Galaxy
  • Editing message properties
  • Adding new entries (useful for custom levels)
  • Escapes sequences supported (color change, icons) [WIP]
  • Graphical user interface
  • Editing from the .arc archive file
